Accordion

Example

This is the first item's accordion body. It is hidden by default, until the collapse plugin adds the appropriate classes that we use to style each element. These classes control the overall appearance, as well as the showing and hiding via CSS transitions. You can modify any of this with custom CSS or overriding our default variables. It's also worth noting that just about any HTML can go within the .accordion-body, though the transition does limit overflow.

HTML

<div class="container">
  <div class"accordion-container">
    <div class="accordion-element">
      <button class="accordion-btn"> Accordion Item #1 </button>
      <div class="accordion-info"> This is the first item's accordion body.</div>
    </div>
    <div class="accordion-element">
      <button class="accordion-btn"> Accordion Item #1 </button>
      <div class="accordion-info"> This is the second item's accordion body.</div>
    </div>
    <div class="accordion-element">
      <button class="accordion-btn"> Accordion Item #1 </button>
      <div class="accordion-info"> This is the third item's accordion body.</div>
    </div>
  </div>
</div>

Sass Variables

    Accordion Container variables
$accordion-container-border-color: $secondary-color;
$accordion-container-width: 100%;
$accordion-container-border: 1px solid $accordion-container-border-color;
$accordion-container-border-radius-top: 4px 4px 0 0;
$accordion-container-border-radius-bottom: 0 0 4px 4px;

    Accordion Button Variables
$accordion-btn-bg-color: $primary-color;
$accordion-btn-color: $white;
$accordion-btn-border: $accordion-container-border;
$accordion-btn-cursor: pointer;
$accordion-btn-padding-x: 10px;
$accordion-btn-padding-y: 20px;
$accordion-btn-font-family: $font-family-base;
$accordion-btn-font-size: $font-size-base;
$accordion-btn-line-height: $line-height-base;
$accordion-btn-letter-spacing: $letter-spacing-base;
$accordion-btn-focus-color: $white;
$accordion-btn-focus-border: 2px solid $accordion-container-border-color;

    Accordion Info variables
$accordion-info-bg-color: $white;
$accordion-info-border: $accordion-container-border;
$accordion-info-padding-x: 10px;
$accordion-info-padding-y: 20px;
$accordion-info-font-family: $font-family-base;
$accordion-info-font-size: 14px;
$accordion-info-line-height: $line-height-base;
$accordion-info-letter-spacing: $letter-spacing-base;
