Box-Shadow

Used in casting shadows (often called “Drop Shadows”, like in Photoshop) from elements. Here is an example with the deepest possible browser support:

HTML

<div class="width-2 height-2 box-shadow-0 background-color-primary"></div>

SCSS

$box-shadows: ();
$box-shadows: map-merge(
  (
    0: 10px 10px 10px 2px map-get($colors, "black"),
    1: 10px 10px 10px 2px map-get($colors, "white"),
  ),
  $box-shadows
);
