Place-Items

The place-items property in CSS is shorthand for the align-items and justify-items properties, combining them into a single declaration.

HTML

<div class="display-grid place-items-center">
  <div class="height-3 width-3 background-color-info"></div>
</div>

SCSS

$place-items: ();
$place-items: map-merge(
  (
    "auto": auto,
    "normal": normal,
    "stretch": stretch,
    "start": start,
    "end": end,
    "center": center,
    "left": left,
    "right": right,
    "flex-start": flex-start,
    "flex-end": flex-end,
    "self-start": self-start,
    "self-end": self-end,
    "first": first,
    "baseline": baseline,
    "last": last,
  ),
  $place-items
);
