Grid-Template-Columns / Grid-Template-Rows

The grid-template-rows and grid-template-columns properties are the primary CSS properties for establishing a grid layout, once the element is a grid context (display: grid;).

HTML

<div class="display-grid grid-template-row-example grid-template-column-example gap-20px height-4">
  <div class="background-color-info min-width-100px grid-row-example-1 grid-row-example-1 display-grid place-items-center"></div>
  <div class="background-color-info min-width-100px grid-row-example-2 grid-row-example-2 display-grid place-items-center"></div>
  <div class="background-color-info min-width-100px grid-row-example-3 grid-row-example-3 display-grid place-items-center"></div>
</div>

SCSS

$grid-template-rows: ();
$grid-template-rows: map-merge(
  (
    "example": repeat(3, 1fr),
  ),
  $grid-template-rows
);

$grid-template-columns: ();
$grid-template-columns: map-merge(
  (
    "example": repeat(3, 1fr),
  ),
  $grid-template-columns
);
