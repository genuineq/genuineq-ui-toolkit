Font

The font property in CSS is a shorthand property that combines all the following sub-properties in a single declaration.

Font-size: 28px

Font-family: verdana

Font-weight: 700

HTML

<p class="font-size-3">Font-size: 28px</p>
<p class="font-family-verdana">Font-family: verdana</p>
<p class="font-weight-3">Font-weight: 700</p>
