Height

The height property in CSS defines specifies the content height of boxes and accepts any of the length values.

height-1
height-2
height-3
height-4

<div class="height-1 width-1 background-color-info display-grid place-items-center">height-1</div>
<div class="height-2 width-1 background-color-info display-grid place-items-center">height-2</div>
<div class="height-3 width-1 background-color-info display-grid place-items-center">height-3</div>
<div class="height-4 width-1 background-color-info display-grid place-items-center">height-4</div>
