Width

The width property in CSS specifies the width of the element’s content area. This “content” area is the portion inside the padding, border, and margin of an element (the box model).

width: 200px;

width: 100%;

width: 300px;

HTML

<div class="width-3 height-2 background-color-info mb-7 display-grid place-items-center">
  <p>width: 200px;</p>
</div>
<div class="width-100 height-2 background-color-info mb-7 display-grid place-items-center">
  <p>width: 100%;</p>
</div>
<div class="width-300px height-2 background-color-info display-grid place-items-center">
  <p>width: 300px;</p>
</div>
