User-Select

The user-select property in CSS controls how the text in an element is allowed to be selected. For example, it can be used to make text unselectable.

You can't copy me.

You can copy me like any other text from a website.

One click selects all this.

HTML

<div class="height-1 background-color-info display-grid place-items-center mb-5">
  <p class="user-select-none">You can't copy me.</p>
</div>
<div class="height-1 background-color-info display-grid place-items-center mb-5">
  <p>You can copy me like any other text from a website.</p>
</div>
<div class="height-1 background-color-info display-grid place-items-center">
  <p class="user-select-all">One click selects all this.</p>
</div>

SCSS

$user-select: ();
$user-select: map-merge(
  (
    "none": none,
    "all": all,
    "auto": auto,
  ),
  $user-select
);
