Transform

The transform property allows you to visually manipulate an element by skewing, rotating, translating, or scaling:

HTML

<div class="animation-name-skew animation-duration-3s animation-iteration-count-infinite background-color-info height-2 width-1 animation-direction-alternate transform-example-1"></div>

SCSS

$transform: ();
$transform: map-merge(
  (
    0: translateY(0),
    1: translateY(-100%),
    "flip": rotateY(180deg),
    "example-1": skew(20deg),
  ),
  $transform
);

$transform-origin: ();
$transform-origin: map-merge(
  (
    "top-left": top left,
  ),
  $transform-origin
);

$transform-style: ();
$transform-style: map-merge(
  (
    "preserve-3d": preserve-3d,
    "flat": flat,
  ),
  $transform-style
);
