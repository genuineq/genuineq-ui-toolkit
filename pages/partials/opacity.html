Opacity

The opacity property in CSS specifies how transparent an element is.

opacity: 1;

opacity: 0.7;

opacity: 0.5;

opacity: 0.1;

HTML

<div class="opacity-1 height-2 width-1 background-color-info display-grid place-items-center m-3">
  <p>opacity: 1;</p>
</div>
<div class="opacity-07 height-2 width-1 background-color-info display-grid place-items-center m-3">
  <p>opacity: 0.7;</p>
</div>
<div class="opacity-05 height-2 width-1 background-color-info display-grid place-items-center m-3">
  <p>opacity: 0.5;</p>
</div>
<div class="opacity-01 height-2 width-1 background-color-info display-grid place-items-center m-3 ">
  <p>opacity: 0.1;</p>
</div>

SCSS

$opacity: ();
$opacity: map-merge(
  (
    "1": 1,
    "07": 0.7,
    "05": 0.5,
    "03": 0.1,
    "01": 0.1,
  ),
  $opacity
);
