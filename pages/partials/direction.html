Direction

The direction property in CSS sets the direction of of content flow within a block-level element. This applies to text, inline, and inline-block elements. It also sets the default alignment of text and the direction that table cells flow within a table row.

Direction: rtl

Direction: ltr

HTML

<p class="direction-rtl background-color-primary color-white p-3 mb-3">Direction: rtl</p>
<p class="direction-ltr background-color-primary color-white p-3">Direction: ltr</p>

SCSS

$direction: ();
$direction: map-merge(
  (
    "ltr": ltr,
    "rtl": rtl,
    "inherit": inherit,
  ),
  $direction
);
