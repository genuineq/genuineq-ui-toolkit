Backdrop-Filter

The backdrop-filter property in CSS is used to apply filter effects (grayscale, contrast, blur, etc) to the background/backdrop of an element. The backdrop-filter has the same effect as the filter property, except the filter effects are applied only to the background and instead of to the elementâ€™s content.

The background image should be pleasantly blurred under this box.

HTML

<div class="background-image-1">
  <div class="backdrop-filter-10px">
    <p>The background image should be pleasantly blurred under this box.</p>
  </div>
</div>

SCSS

$backdrop-filter: ();
$backdrop-filter: map-merge(
  (
    "grayscale": grayscale,
    "contrast": contrast,
    "blur": blur,
    "10px": blur(10px),
  ),
  $backdrop-filter
);
