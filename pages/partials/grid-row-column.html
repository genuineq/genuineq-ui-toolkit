Grid-Column / Grid-Row

The grid-row and grid-column properties define which row or column an element will be displayed on.

grid-column: 1 / 3; grid-row: 2 / 3;
grid-column: 3 / -1; grid-row: 1 / 2;
grid-column: 3 / 4; grid-row: 2 / 4;

HTML

<div class="background-color-info min-width-100px grid-row-example-1 grid-row-example-1 display-grid place-items-center"> grid-column: 1 / 3; grid-row: 2 / 3;</div>
<div class="background-color-info min-width-100px grid-row-example-2 grid-row-example-2 display-grid place-items-center">grid-column: 3 / -1; grid-row: 1 / 2;</div>
<div class="background-color-info min-width-100px grid-row-example-3 grid-row-example-3 display-grid place-items-center">grid-column: 3 / 4; grid-row: 2 / 4;</div>

SCSS

$grid-row: ();
$grid-row: map-merge(
  (
    "12.5rem": 12.5rem,
    "example-1": 2/3,
    "example-2": 1/2,
    "example-3": 2/4,
  ),
  $grid-row
);

$grid-column: ();
$grid-column: map-merge(
  (
    "example-1": 1/3,
    "example-2": 3/-1,
    "example-3": 3/4,
  ),
  $grid-column
);
