Display

Every element on a web page is a rectangular box. The display property in CSS determines just how that rectangular box behaves. There are only a handful of values that are commonly used:

Display: none
Display: inline
Display: inline-block
Display: block
Display: run-in

Display: Flex

HTML

<div class="display-none height-2 width-1 background-color-info">Display: none</div>
<div class="display-inline height-2 width-1 background-color-info ml-3">Display: inline<div>
<div class="display-inline-block height-2 width-1 background-color-info mb-3">Display: inline-block</div>
<div class="display-block height-2 width-1 background-color-info mb-3">Display: block</div>
<div class="display-run-in height-2 width-1 background-color-info mb-3">Display: run-in</div>
<div class="display-flex justify-content-space-around align-items-space-around flex-wrap-wrap height-2 width-1 background-color-info">
  <p class="width-1">Display: Flex</p>
  <div class="height-20px width-20px background-color-black"></div>
  <div class="height-20px width-20px background-color-black"></div>
  <div class="height-20px width-20px background-color-black"></div>
  <div class="height-20px width-20px background-color-black"></div>
</div>

SCSS

$display: ();
$display: map-merge(
  (
    "none": none,
    "inline": inline,
    "inline-block": inline-block,
