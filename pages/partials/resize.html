Resize

The resize property controls if and how an element can be resized by the user by clicking and dragging the bottom right corner of the element.

HTML

<textarea class="max-width-780px resize-both" cols="30" rows="10"></textarea>

SCSS

$resize: ();
$resize: map-merge(
  (
    "none": none,
    "both": both,
    "horizontal": horizontal,
    "verical": vertical,
  ),
  $resize
);
