Caret

The caret-color property in CSS changes the color of the cursor (caret) in inputs, texareas, or really any element that is editable.

The caret-shape property in CSS changes the shape of the text cursor inside editable elements that indicates a user is typing. Itâ€™s part of the CSS Basic User Interfaces Module Level 4, which is currently in Working Draft status.
