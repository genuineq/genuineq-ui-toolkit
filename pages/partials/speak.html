Speak

The speak property in CSS is for specifying if a browser should speak the content it reads, such as through a screen reader.

SCSS

$speak: ();
$speak: map-merge(
  (
    "auto": auto,
    "visible": visible,
    "never": never,
    "always": always,
  ),
  $speak
);

$speak-as: ();
$speak-as: map-merge(
  (
    "normal": normal,
    "spell-out": spell-out,
    "digits": digits,
    "literal-punctuation": literal-punctuation,
    "no-punctuation": no-punctuation,
  ),
  $speak-as
);
