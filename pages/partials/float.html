Float

The float property in CSS is used for positioning and layout on web pages. A common usage might be floating an image to one side and letting text wrap around it.

HTML

<div class="height-2 width-1 background-color-info float-left"></div>
<div class="height-2 width-1 background-color-info float-right"></div>

SCSS

$float: ();
$float: map-merge(
  (
    "none": none,
    "top": top,
    "bottom": bottom,
    "left": left,
    "right": right,
    "inherit": inherit,
  ),
  $float
);
