Object-Position

The CSS property object-position is used to align any selected replaced element (e.g., an <img>) within the box that contains it.

object-position: 20% 80%;

object-position: 40% 30%;

HTML

<div class="display-grid place-items-center height-300px">
  <div class="width-300px height-4 background-color-info border-style-solid border-width-10px border-color-primary">
    <img class="width-3 object-position-20p-80p" src="https://genuineq.com/wp-content/themes/genuineq/assets/images/genuineq-og-image.jpg" alt="">
  </div>
  <p>object-position: 20% 80%;

</div>
