Tab-Size

The tab-size property in CSS is used to adjust the amount of spaces that display for the tab character.

  TAB-SIZE: 8; Default Value 
a {
  color: red;
  &:hover,
  &:focus {
    color: blue;
  }
}

  TAB-SIZE: 2;
a {
  color: red;
  &:hover,
  &:focus {
    color: blue;
  }
}

  TAB-SIZE: 20px;
a {
  color: red;
  &:hover,
  &:focus {
    color: blue;
  }
}
