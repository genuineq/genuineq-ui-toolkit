Perspective

The perspective CSS property gives an element a 3D-space by affecting the distance between the Z plane and the user.

HTML

SCSS

$perspective: ();
$perspective: map-merge(
  (
    "none": none,
    1000: 1000px,
  ),
  $perspective
);

$perspective-origin: ();
$perspective-origin: map-merge(
  (
    0: 50% 30%,
  ),
  $perspective-origin
);
