Position

The position property can help you manipulate the location of an element, for example:

position: relative;

top: 20px;

left: 50%;

HTML

<div class="position-relative height-3 width-3 background-color-info top-20px left-50p d-flex flex-direction-column justify-content-center flex-wrap-wrap align-items-center">
  <p>position: relative;</p>
  <p>top: 20px;</p>
  <p>left: 50%;</p>
</div>

SCSS

$position: ();
$position: map-merge(
  (
    "static": static,
    "relative": relative,
    "absolute": absolute,
    "fixed": fixed,
    "sticky": sticky,
  ),
  $position
);
