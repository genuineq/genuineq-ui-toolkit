Max-Height

The max-height property in CSS is used to set the maximum height of a specified element. Authors may use any of the length values as long as they are a positive value. max-height overrides height, but min-height always overrides max-height.

max-height: 300px;

HTML

<div class="height-400px width-4 background-color-info max-height-300px display-grid place-items-center">
  <p>max-height: 300px;</p>
</div>

SCSS

$max-height: ();
$max-height: map-merge(
  (
    "100px": 100px,
    "300px": 300px,
  ),
  $max-height
);

Max-Width

The max-width property in CSS is used to set the maximum width of a specified element. The max-width property overrides the width property, but min-width will always override max-width whether followed before or after width in your declaration. Authors may use any of the length values as long as they are a positive value.

max-width: 300px;

HTML

<div class="height-3 width-600px background-color-info max-width-300px display-grid place-items-center">
  <p>max-width: 300px;</p>
</div>

SCSS

$max-width: ();
$max-width: map-merge(
  (
    "100px": 100px,
    "300px": 300px,
    "400px": 400px,
  ),
  $max-width
);
