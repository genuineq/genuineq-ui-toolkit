Color

The color property in CSS sets the color of text and text decorations.

color-primary
color-warning
color-info

HTML

<div class="color-primary">color-primary</div>
<div class="color-warning">color-warning</div>
<div class="color-info">color-info</div>

SCSS

@each $attribute, $name in (color: color) {
  @each $key, $value in $colors {
    .#{$name}-#{$key} {
      #{$attribute}: $value;
    }
  }
  }
