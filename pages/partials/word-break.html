Word-Break

The word-break property in CSS can be used to change when line breaks ought to occur. Normally, line breaks in text can only occur in certain spaces, like when there is a space or a hyphen.

example

word-break-break-all

HTML

<h1 class="word-break-break-all text-transform-uppercase font-weight-bold width-1em">example</h1>
<p class="font-size-1 text-align-center width-100 mt-4">word-break-break-all</p>

SCSS

$word-break: ();
$word-break: map-merge(
  (
    "break-all": break-all,
    "normal": normal,
    "keep-all": keep-all,
  ),
  $word-break
);
